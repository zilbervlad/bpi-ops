Printer Setup Steps

Use the same Bluetooth flow that already works today.

  1. Open this page on the store computer or phone.
  2. Scan the Android Printer App QR with the Android device.
  3. Install or open the BPI Labels app.
  4. Pair the Android device to the Zebra / label printer through Bluetooth.
  5. Scan the Store Setup QR so the app knows which store it belongs to.
  6. Print labels from the Android app.
